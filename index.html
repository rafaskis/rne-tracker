Google Drive Cloud Sync

You can securely push your database direct to the cloud without intermediate files:

  • Clicking "Sync Data to Google Drive" triggers an authorization pop-up window.
  • The app strictly uses isolated drive.file tracking, giving it access only to items it generates itself.
  • Files save with clear date tags: RNE_YYMMDD_HHMM.txt.

Smart Group Visibility Modes

Categories with **unpaid balances** expand automatically by default to catch your attention. Categories that are **fully completed** collapse neatly so you can focus only on what's left.

App User Guide

Discreet Cloud Control Bar

Located at the top of the interface for convenience without clutter:

  • Click "Sign In" to securely link your Google Profile session.
  • Once authorized, dedicated inline buttons allow you to immediately push a new Backup or run a file Restore.

Automatic Overdue Month-Rollover

The system actively runs an audit on startup. If the current calendar month has advanced past your open tracking view, any lingering unpaid items from the previous month are automatically migrated forward to the new month.

Historical Source Suffixes

Whenever an expense is moved to another month—either manually using the arrow actions or automatically by the system rollover engine—a suffix indicating its original target month (e.g., (7/26) for July 2026) is automatically appended directly to its name tag to preserve context.

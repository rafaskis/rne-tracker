App User Guide

Dedicated Cloud Database

The system creates and maintains a single clean file inside your Google Drive:

  • Clicking "Connect Account & Sync" logs you in and immediately searches for your database file (rne_database.json).
  • Auto-Restore: If the file is found, your local view automatically overwrites with the latest cloud state.
  • Smart Overwrite: Clicking "Save Changes to Cloud File" safely updates that exact same file directly, ensuring your Drive stays perfectly clean.

Smart Group Visibility Modes

Categories with **unpaid balances** expand automatically by default to catch your attention. Categories that are **fully completed** collapse neatly so you can focus only on what's left.
